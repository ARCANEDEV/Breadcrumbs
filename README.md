    - PHP >= 5.4.0

# INSTALLATION

## Composer
You can install this package via [Composer](http://getcomposer.org/). Add this to your `composer.json` :

```json
{
    "require": {
        "arcanedev/breadcrumbs": "~1.0"
    }
}
```    

Then install it via `composer install` or `composer update`.
